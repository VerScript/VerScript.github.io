VerScript Documentation

Welcome to the VerScript documentation. The language is built on a simple, spaced-function syntax that natively supports console IO, variable assignment, and mathematical evaluation.

1. The display Keyword

Output text, numbers, or variables directly to the terminal.

display "Hello World" display 100

2. Variables (var: val)

Variables are dynamically typed strings, integers, or booleans. Assign them using a colon.

name: "Alice" age: 25 is_active: true display name

3. Mathematical Operators & String Concatenation

VerScript supports standard arithmetic (+, -, *, /) seamlessly, as well as negative numbers and string concatenation.

score: 10 + -5 display "Score: " + score display 100 / 4

4. The prompt Keyword

Make your terminal scripts interactive by asking the user for input.

display "Enter your name: " prompt username display "Welcome, " display username

5. The loop Keyword

Repeat a block of code n times. The block of code must be indented using standard spacing (e.g. 2 spaces or a tab).

loop 3 display "This repeats 3 times"

6. The iterate Keyword

Iterate an index variable between specified numeric limits. You can customize the iteration index or omit the start limit (which defaults to the variable's current value, or 0 if undefined).

! Full syntax (inclusive limits): iterate i from 1 to 5 display i ! Omitted start limit (starts at current value of variable, or 0 if undefined): count: 3 iterate count to 6 display count

7. Comparison Operators

VerScript supports standard comparison operators that evaluate to booleans (true / false):

  • = (Equal to, equivalent to ==)
  • x= (Not equal to, equivalent to !=)
  • > (Greater than)
  • < (Less than)
  • >= (Greater than or equal to)
  • <= (Less than or equal to)
is_match: 10 = 10 display is_match ! Prints true is_diff: 5 x= 10 display is_diff ! Prints true

8. Conditionals (if-else)

Control flow using standard conditional statements. Like loops, conditional branches execute indented blocks of code.

val: 15 if val = 10 then display "Value is exactly 10" else if val > 10 then display "Value is greater than 10" else display "Value is less than 10"

9. Looping Conditionals (while & until)

Execute statement blocks dynamically based on expression checks:

  • while: Repeatedly runs while the condition evaluates to true.
  • until: Repeatedly runs until the condition evaluates to true (i.e. while it is false).
! While loop example count: 1 while count <= 3 display "count: " + count count: count + 1 ! Until loop example (runs until count2 is greater than 3) count2: 1 until count2 > 3 display "until count2: " + count2 count2: count2 + 1

10. Exception Handling & Conditional Fallbacks (do-unless)

The do-unless block has two forms of execution:

Exception Handling (Try-Catch)

Catch runtime errors thrown by the compiler/interpreter. Errors can be caught by the generic keyword error or specific error types like DivisionByZeroError, UndefinedVariableError, InvalidOperandError, IndentationError, LoopIterationError, LoopLimitError, LoopDirectionError, or SyntaxError.

do display "Trying division by zero..." val: 10 / 0 unless DivisionByZeroError display "Oops, divided by zero!" do display "Accessing undefined variable..." display missing_val unless error display "Caught a general error!"

Conditional Execution

Execute fallback logic if a boolean expression evaluates to true, otherwise run the primary block.

flag: true do display "This executes if flag is false" unless flag display "This executes if flag is true"
