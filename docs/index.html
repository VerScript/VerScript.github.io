VerScript Documentation

Welcome to the VerScript documentation. The language is built on a simple, spaced-function syntax that natively supports console IO, variable assignment, and mathematical evaluation.

1. The display Keyword

Output text, numbers, or variables directly to the terminal.

display "Hello World" display 100

2. Variables (var: val)

Variables are dynamically typed strings or integers. Assign them using a colon.

name: "Alice" age: 25 display name

3. Mathematical Operators

VerScript supports standard arithmetic (+, -, *, /) seamlessly.

score: 10 + 5 display score display 100 / 4

4. The prompt Keyword

Make your terminal scripts interactive by asking the user for input.

display "Enter your name: " prompt username display "Welcome, " display username

5. ✨ VS# AI Assistant

Enhance your development with VS#, a smart AI assistant powered by Gemini. Click the ✨ VS# button in the IDE header to open the AI panel.

Ask VS# to generate code, explain snippets, or fix syntax errors. When VS# applies a correction or writes new code, it integrates directly with the editor. You will see a live backspacing and typing animation that shows the code modifying in real-time right before your eyes.
