Lightweight C Core

VerScript is built on a minimal C foundation. It includes a custom lexer and interpreter designed for fast tokenization.

Dynamic Types & Expressions

Enjoy dynamic typing with support for strings, integers, and booleans. Effortlessly concatenate variables and evaluate mathematical expressions directly within the language.

🔄 Loops & Iterations

Supports clean Python-like indented loop structures (loop n) and variable iteration bounds (iterate i from x to y) compiled and executed on the fly.

Spaced Function Syntax

Functions and arguments are evaluated via space separation, promoting a remarkably clean and readable text aesthetic.

✨ Custom VS# LLM

Equipped with a custom backprop neural network helper built completely from scratch. Edits backspace and type out dynamically inside the Monaco Editor.
