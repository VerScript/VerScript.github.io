Enter Documentation Open Web IDE

Lightweight C Core

VerScript is built on a minimal C foundation. It includes a custom lexer and interpreter designed for fast tokenization.

Strict String Typing

The "display" keyword enforces strict string-based outputs, allowing you to easily push text to the console directly from the VM.

Spaced Function Syntax

Functions and arguments are evaluated via space separation, promoting a remarkably clean and readable text aesthetic.

✨ Smart VS# AI

Equipped with a live Gemini LLM to write, modify, and fix code. Interactive edits are animated directly in your editor with typing animations.
